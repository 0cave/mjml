## mjml-spacer

```xml
<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>
          <mj-spacer height="50px" />
        <mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```

Displays a blank space.

attribute                   | unit        | description                    | default value
----------------------------|-------------|--------------------------------|------------------------------
height                      | px          | spacer height                  | 20px
