## MJ-CONTAINER

<aside class="notice">
  MJ-Container was MJ-Body in 1.X
</aside>

``` html

<mjml>
  <mj-body>
    <mj-container>

      <!-- You email goes here -->

    </mj-container>
  </mj-body>
</mjml>

```

This is the starting point of your email.
